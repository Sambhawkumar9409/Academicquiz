📚 Academic Quiz App

🎯 Think you know your Science, Maths, and English well enough?
Put your knowledge to the test — you've got just 30 seconds per question. No cheating. No second chances. Just you, the clock, and your brain. ⏱️🧠

A fast-paced, interactive quiz app built for students who want to challenge themselves beyond the classroom. With instant feedback, a live countdown timer, and a detailed score breakdown at the end — this isn't just a quiz, it's a learning experience.
🚀 Live Demo
🔗 sambhawacademicquiz.netlify.app
✨ Features

🧠 10 Questions covering Science, Maths & English
⏱️ 30-second timer per question with live countdown
✅ Instant feedback after every answer
📊 Subject-wise score breakdown at the end
🏆 Grading system — Excellent / Good / Needs Improvement / Keep Practicing
📱 Fully responsive — works on mobile, tablet & desktop

🛠️ Tech Stack

Frontend: React.js (with Hooks)
Styling: Inline CSS with modern UI design
Deployment: Netlify
Build Tool: Babel (CDN-based, no build step required)
